HTML Tags


The basic syntax for a tag would be...

<video,audio src="filename"></video,audio>
You can also add quite a few controls through HTML like height, width, etc by adding the parameters after the file name.
There is also the autoplay fearure that can be added. In most cases this is just a really obnoxius way to force the content
on the end user as by itself it removes controls. If you really want to irritate the end user you can add the loop feature
as well.
There are times when these features are usefull, for example in the video on this page I set it to autoplay on a loop, but
this video has no sound, and is not very distracting. you can also add a conrols feature to allow end users to pause features
I have also added this feature to this video.

This is the line for this short video <video src="Toy_3.mp4" width="800" height="800" autoplay loop controls></video>


Canvas is a little bit different from other HTML tags. Canvas allows the use of JavaScript in a window for many different applications.
You can create mini games, animations, or pictures, as well as other things in JavaScript and then pull them into the HTML canvas window
Instead of using source="filename" you would use id="objectname" Other than that the syntax is the same.
