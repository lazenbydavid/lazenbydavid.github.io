Loops

Loops are handy if you need to run a piece of code over and over again. This can be used for a counting function,
or to display something like a inventory, or many other uses. The most common is the for statement.
If I were to write <code> var text = ""; var i; for (i = 0; i < 5; i++) { text += "the number is " + i + "<br>"; } document.getElementById("counting").innerHTML = text; Then it would produce a counter like this.



Conditional Statements

Conditional statements set up a list of conditions that if met will execute the command for that line. They
start with an "if", and can run through more condtions with "else if". It is good practice to finish a conditional
statement with a final "else" statement with no condtion to be met, this can output an error message or code
letting the user know that none of the condtions were met.
an example of this will be on the next section for functions.

Variables

Varibles are basic and used everywhere. To create a variable you simply make the name of the variable and give
it a value. You can change the value of the variable by using many of the different methods described in this page.
Example - var a = 27. In this example if you used the variable "a" it would give you the value of 27.

Arrays & Associate Arrays

Arrays are variables with multiple values attached in an index. An associative array uses a named index verses
calling the valuses by number. The important thing to remember when using an array is that the first value in the
array is called with a 0. So if you have an array of "var fruit = ["pear", "Pinapple", "Orange"]; Pear would be
called with a 0, and Pinapple would be called with a 1 when accessing the array.

Object Creation Function

We know how functions work from the earlier section. You can also create objects with a function by giving the
function a list of parameters then using the this keyword, then populating them with variables using the "new"
keyword. You can then use those variables with the values that have been added. Lets see how this works.
<code> function house(where, bedroom, bathroom) { this.town = where; this.rooms = bedroom; this.bath = bathroom; } var beachhouse = new house("Corona", 3, 2); var townhouse = new house("Boise", 4, 2) </code> So with this function I am creating two new objects with my variables, a beach house in Corona with 3 bedrooms and
2 bathrooms, and a townhouse in Boise, with 4 bedrooms and 2 bathrooms.

Inheritance

Inheritance works when an object is nested within another object inheriting traits from the parent object. In our last
example we have the "where" object that tells us what city the house is in. if we want to break that down a little more
we can use the this keyword inside the where to create a further breakdown. For example.
<code> function house(where, bedroom, bathroom) { this.town = { city, state }; this.rooms = bedroom; this.bath = bathroom; } </code> So with this change we can now call "this.town" to call both city, and state, or if we want to break it down further we can
call just city with "this.town.city"
